## 4. Lighthouse results

**Not re-run for this pass.** Justification:

- The theming refactor is **pure presentation**: no new runtime
  JavaScript, no new server endpoints, no new content fetches.
- Fraunces is self-hosted by `next/font` as a build-time asset
  (preloaded, ~50 kB woff2). It does not introduce a runtime CDN
  request or CLS; the preload header makes it a known-cost addition,
  not a regression vector.
- The `@theme` token projection adds a small amount of generated CSS
  (~16 token names × utility families) but the actual rendered CSS
  payload is dominated by Tailwind's main bundle, not by the new
  tokens.

The 2026-05-29 pass had every category hitting its ADR-0004 floor.
ADR-0004 floors remain unchanged. **Recommended next pass**: re-run
Lighthouse against the post-FEAT-012 build, then again after A1 lands
real content. Two re-runs are enough to track the curve through v1
launch.
